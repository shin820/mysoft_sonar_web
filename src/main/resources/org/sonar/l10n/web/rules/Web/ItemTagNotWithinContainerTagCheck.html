Using a <li> or <dt> item tag outside of a <ul>, <ol> or <dl> one does not make sense and indicates a bug.

The following code snippet:

<li>Apple</li>          <!-- Non-Compliant -->
<li>Strawberry</li>     <!-- Non-Compliant -->

<li>Apple</li>          <!-- Non-Compliant -->
<li>Strawberry</li>     <!-- Non-Compliant -->

<dt>Apple</dt>          <!-- Non-Compliant -->
<dt>Strawberry</dt>     <!-- Non-Compliant -->

should be refactored into:

<ul>
  <li>Apple</li>        <!-- Compliant -->
  <li>Strawberry</li>   <!-- Compliant -->
</ul>

<ol>
  <li>Apple</li>        <!-- Compliant -->
  <li>Strawberry</li>   <!-- Compliant -->
</ol>

<dl>
  <dt>Apple</dt>        <!-- Compliant -->
  <dt>Strawberry</dt>   <!-- Compliant -->
</dl>
